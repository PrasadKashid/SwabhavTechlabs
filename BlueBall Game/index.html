Welcome to the Number Guessing Game

Game Rules

  • You have to guess a number between 1 to 40 in 6 Attempts
  • If you choose a correct number then, the circle turns Green.
  • If you choose a number Smaller than the generated number, the circle turns Yellow.
  • If you choose a number Larger than the generated number, the circle turns Red.

    • Click play to start the game!
